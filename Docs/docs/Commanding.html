Commanding

 

Commanding is the same as normal prism Commands.  Some of the standard commands will be handled by default in the AppCanApp base class.  Those can be overridden if desired.  An example of the commanding can be seen by looking at the Samples\SimpleApp

 

Open, Save, SaveAs, Close standard commands are handled.  AppCan adds AppCanCommands which at the moment includes an Exit command.

 

Future ideas:

Future thoughts on commanding would be to support an AppCan command that can take a context name and it would create that context.  This would allow creating a window directly through a command from a context.
