Installation

 

Just download the code and put it into a directory. i.e. something like d:\AppCan

For samples you will find the directories:

Libs

Samples

 

The Libs directory has the AppCan, Prism and log4net binaries.

The samples directory has examples of using AppCan.

 

For the full code base it will contain the folders:

AppCan

Libs

Samples

 

The AppCan directory contains the AppCan source code.
